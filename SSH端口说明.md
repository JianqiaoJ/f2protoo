# SSH 端口 22 说明

## 🤔 什么是 SSH？

**SSH (Secure Shell)** 是一种远程登录服务器的协议，就像用钥匙开门一样，让您可以从本地电脑连接到远程服务器。

## 📡 为什么需要 SSH？

当您执行以下操作时，实际上都在使用 SSH：

1. **使用 `ssh ubuntu@43.143.53.202` 登录服务器**
2. **使用 `scp` 命令上传文件到服务器**
3. **使用 `deploy.sh` 脚本自动部署**（脚本内部使用SSH）

## 🔌 端口 22 是什么？

- **端口 22** 是 SSH 的**默认端口**
- 就像门牌号一样，告诉系统"这是SSH服务的入口"
- 当您使用 `ssh` 或 `scp` 命令时，**自动连接到端口 22**

## 💡 为什么会出现"连接超时"？

当我说"SSH 端口 22 连接超时"时，意思是：

```
您的电脑 → 尝试连接 → 服务器的端口22 → ❌ 被阻止/无法连接
```

**可能的原因：**
1. 服务器的防火墙关闭了端口 22
2. 云服务商的安全组规则没有开放端口 22
3. SSH 服务没有运行

## ✅ 解决方案

### 方案 1: 不需要 SSH（推荐）

**直接在腾讯云控制台操作**，不需要SSH：

1. 点击控制台的 **"登录"** 按钮
2. 在浏览器中打开终端
3. 直接执行部署命令

**这样就不需要SSH连接了！**

### 方案 2: 修复 SSH（如果想用命令行）

如果您想使用 `ssh` 命令或自动部署脚本，需要：

1. **在腾讯云控制台** → **"防火墙"** 标签页
2. **添加规则**：开放端口 22
3. 然后就可以使用 SSH 了

## 🎯 总结

- **SSH 端口 22** = 远程登录服务器的"门"
- **连接超时** = 这个"门"被关闭了
- **解决方案** = 直接在控制台操作，不需要SSH

## 📋 推荐操作

**不需要修复SSH端口**，直接：

1. 在腾讯云控制台点击 **"登录"** 按钮
2. 在打开的终端中执行部署命令
3. 或者使用 **"文件管理"** 功能上传文件

这样就不需要SSH连接了！
