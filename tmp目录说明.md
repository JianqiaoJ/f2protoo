# /tmp/ 目录说明

## 📍 /tmp/ 目录在哪里？

`/tmp/` 是 **Linux 系统的临时目录**，位于**根目录**下。

### 目录结构：

```
/                    ← 根目录（系统最顶层）
├── tmp/             ← 临时文件目录（这就是我们要用的）
├── var/
├── opt/
├── home/
├── usr/
└── ...
```

## 🎯 为什么使用 /tmp/ 目录？

1. **临时存储**：用于存放临时文件
2. **所有用户可访问**：任何用户都可以读写
3. **系统重启可能清空**：但通常不会立即清空
4. **适合部署**：上传文件后可以解压和部署

## 💡 在腾讯云控制台中如何找到？

### 方法 1: 使用文件管理

1. 点击控制台的 **"文件管理"** 标签页
2. 在文件浏览器中，导航到根目录 `/`
3. 找到 `tmp` 文件夹
4. 进入 `tmp` 文件夹，就是 `/tmp/` 目录

### 方法 2: 使用终端

1. 点击控制台的 **"登录"** 按钮
2. 在终端中输入：
   ```bash
   cd /tmp
   pwd    # 显示当前路径，应该显示 /tmp
   ls     # 查看目录内容
   ```

## 📋 实际操作步骤

### 上传文件到 /tmp/ 目录：

1. **使用文件管理**：
   - 进入 `/tmp/` 目录
   - 点击"上传"按钮
   - 选择 `deploy-package.tar.gz` 文件
   - 上传完成

2. **使用终端**（如果已上传到其他位置）：
   ```bash
   # 如果文件在其他位置，移动到 /tmp/
   mv /path/to/deploy-package.tar.gz /tmp/
   
   # 或者直接上传到 /tmp/
   # （通过文件管理功能）
   ```

## 🔍 验证文件是否在 /tmp/ 目录

在终端中执行：

```bash
# 进入 /tmp/ 目录
cd /tmp

# 查看文件是否存在
ls -lh deploy-package.tar.gz

# 如果看到文件，说明上传成功
```

## 📝 完整路径示例

- **完整路径**：`/tmp/deploy-package.tar.gz`
- **在终端中**：`cd /tmp` 然后 `ls` 就能看到文件
- **在文件管理中**：导航到 `/` → `tmp` → 就能看到文件

## 💡 提示

- `/tmp/` 是**绝对路径**，从根目录开始
- 在终端中，`cd /tmp` 可以直接进入这个目录
- 在文件管理中，需要从根目录 `/` 开始导航
